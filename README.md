# Todo API [![CI Status](https://travis-ci.org/hyanmandian/todo-api.svg?branch=master)](https://travis-ci.org/hyanmandian/todo-api/)
[Go to API!](https://uniritter-todo-api.herokuapp.com/)

## Endpoints
```
GET /tasks
GET /tasks/:task
POST /tasks
PUT /tasks/:task
DELETE /tasks/:task
```
